# Off-Leash Dog Run

Open daily from 5:30 a.m. to 10 p.m.

<p style="grid-column-start: 2; max-width: 33em;" markdown="1">
Grand Park’s dog run features shade, benches, doggie bags and a water fountain for your four-footed friends!
</p>
